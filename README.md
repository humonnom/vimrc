# vimrc  
vimrc settings.  

## Step 1  
**Copy** vimrc file -> **Paste** on your home directory (Name : .vimrc)  

## Step 2  
Open .vimrc file.  
**:w** (Save)  
**:source %** (Renewal)  

## Step 3  
**:PluginInstall** (Plugin install command)
