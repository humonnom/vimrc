# vimrc

vimrc settings.

#### Step 1

-   **Copy** vimrc file.

```
cd ~
git clone https://github.com/humonnom/vimrc.git
mv vimrc/vimrc ./.vimrc
```

-   **Paste** on your home directory.

```
vi .vimrc
```

#### Step 2

-   Open .vimrc file.
-   **:w** (Save)
-   **:source %** (Renewal)

#### Step 3

-   **:PluginInstall** (Plugin install command)
